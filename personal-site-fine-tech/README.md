# Personal Website — Fine Tech bilingual template

This static site is ready for GitHub Pages or Cloudflare Pages.

## What is included

- English pages at the root: `index.html`, `about.html`, `career.html`, `projects.html`, `journal.html`
- French pages in `/fr/`: `index.html`, `a-propos.html`, `parcours.html`, `projets.html`, `journal.html`
- Article templates in both languages
- Hreflang tags on every page
- Canonical URLs on every page
- Open Graph tags
- Basic Person schema on the homepage
- `sitemap.xml`, `robots.txt`, `.nojekyll`
- Fine tech responsive design in `assets/css/styles.css`

## Before publishing

1. Replace `https://your-domain.example` in all files with your real site URL.
   - For a user GitHub Pages site: `https://username.github.io`
   - For a project GitHub Pages site: `https://username.github.io/repository-name`
   - For a custom domain: `https://yourdomain.com`

2. Replace `[Your Name]` and `[Votre Nom]` with your real name.

3. Replace `hello@example.com`, the LinkedIn URL and the `YN` initials in the logo with your real contact details and initials.

4. Upload all files to your GitHub repository.

5. Make sure `index.html` is at the root of the repository.

## Recommended URL structure

- English homepage: `/`
- French homepage: `/fr/`
- English About: `/about.html`
- French About: `/fr/a-propos.html`
- English Career: `/career.html`
- French Career: `/fr/parcours.html`
- English Projects: `/projects.html`
- French Projects: `/fr/projets.html`
- English Journal: `/journal.html`
- French Journal: `/fr/journal.html`

## Content strategy

Start by completing the homepage, About and Career pages. Then add 3 strong projects. After that, publish one article per month in the Journal.
